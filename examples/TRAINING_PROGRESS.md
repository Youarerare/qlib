## 🔍 如何查看进度

### 方法1：定期检查（推荐）

```bash
# 每5分钟运行一次
python simple_status.py
```

**能看到**：
- 进程数
- GPU使用情况
- 最后活动时间

### 方法2：重新运行并显示进度

如果想知道详细的epoch进度和loss值：

```bash
# 先停止当前训练
taskkill /F /IM python.exe

# 然后运行（会显示详细进度）
python run_with_progress.py
```

**能看到**：
- 实时训练输出
- Epoch进度
- Loss值
- IC指标

### 方法3：等待完成后查看结果

```bash
# 训练完成后
pip install mlflow
mlflow ui

# 浏览器打开 http://localhost:5000
```
