**关于ICIR Label的明确说明**：

✅ **Label定义是正确的！**

```python
# 标准做法（代码中已正确实现）
raw_returns = df.groupby(level='instrument')['close'].pct_change()
future_returns = raw_returns.shift(-1)  # 关键：shift(-1)
```

**解释**：
- 因子值在时间T计算
- Label是T到T+1的收益率（未来一期）
- 使用shift(-1)将未来收益率对齐到当前时间
- 这确保了因子值(T)与label(T->T+1)的正确对应关系
